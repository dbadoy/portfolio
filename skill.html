Adoysign



    블록체인을 이용한 전자서명 플랫폼 Adoysign입니다.


    새로운 전자서명 방식을 제시합니다.

    프론트 : Vanila JS　　백엔드 : Nodejs
    블록체인 : EOS　　　문서공유 : IPFS
    DB : MongoDB


    Key Point

    랜덤한 문자열 signCode를 발급, 사용자만 알고 있는 signCode를 이용하여 서명

    ascii-printable ( 95개 ) 중에 랜덤한 문자열을 정한다. signCode는 5글자로 정해져 있으며 변경 가능하다.
    5글자 : 95⁵ = 경우의 수 약 77억
    6글자 : 95⁶ = 경우의 수 약 7350억
    8글자 : 95⁸ = 경우의 수 약 6634조
    ※실제 서비스를 할 경우에는 8글자 이상을 권장, signCode는 외워서 사용하는 것이 아닌 서명하기 전
    발급받는 OTP개념이기 때문에 길어도 문제가 되지 않을 것으로 생각됨.

    특징

    1. EOS를 이용한 전자서명
    2. 개인과 개인간 플랫폼을 통해 자유롭게 문서 등록 및 서명 가능
    3. 다중 서명 가능

    구현 기술

    - 회원가입 - email 인증 ( EOS 계정에 대한 본인 확인 )
    - nodejs express를 이용한 RESTful API 구현
    - IPFS(ipfs-http-client)를 이용한 문서 분산 저장
    - eosjs를 이용한 트랜잭션

포트스캐닝


C

    C WinSock을 이용한 포트스캐닝 및 포트차단리스트 관리 프로그램



    주요기능

    1. 포트스캐닝
    - 범위 지정, 단일 포트 스캐닝
    2. 사용중인 포트 닫기
    - C의 system()을 이용하여 cmd 명령어 사용(netstat), 프로세스의 SID를 출력해준 후 사용자가 선택 종료
    3. 포트차단리스트
    - 차단할 포트번호를 'portlist2.txt'에 저장 후, 차단리스트 실행 하면 해당 번호들의 포트들이 닫힘. 초기화 가능
    (netsh를 이용한 포트 차단)
