    ALIBI

    블록체인을 이용한 아르바이트 경력증명 플랫폼입니다.

    사용자

    회원가입시, 휴대폰 인증을 제공하며 eosjs를 이용하여 EOS계정을 생성

    토큰을 이용한 로그인 구현 ( jwt )

    근로계약서 및 퇴직서 등록

    사업자번호를 입력받아 계약서에 사업체이름을 입력받는 API 구현

    eosjs를 연동하여 계약서의 해시값, 사용자들의 서명값을 블록체인에 저장

    경력증명서 발급 및 공유

    사용자의 경력데이터를 EOS NFT에 넣어 사용자에게 발급

    발급 받은 NFT를 제 3자에게 공유하여 경력데이터를 제출 가능
